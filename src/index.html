Smart Password Manager

Free open source password manager that will save your time storing all your password in a single decentralize location.

Features

This password manager is designed to enable user in any age gap to comfortably use it in addition to its built-in pillars mainly consist of security, multi-platform, and ultimately free to use.

Security

With multiple layer of protection against malware and bot, your password will never be in risk anymore.

Multi-platform

The password manager software has been tested on more than thousand devices. More device will be supported soon. Check our latest supported devices.

Zero Payment
